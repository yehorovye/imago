# imago

simple and efficient image processing library.

## features

imago uses no external libs but pure std.

## roadmap

* [x] bmp support.
* [ ] png support.
* [ ] jpeg support.
* [ ] webp support.
* [ ] avif support.
* [ ] tiff support.
* [ ] ico support.
